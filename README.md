### Installation
1. Clone the repository
```bash
git clone https://github.com/kasikumar-1/spring-boot-sample.git
```
2. Navigate to the project directory
```bash
cd spring-boot-sample
```
3. Run the application
```bash
./mvnw spring-boot:run
```
